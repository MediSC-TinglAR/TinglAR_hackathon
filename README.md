## 💁🏻 서비스 소개
어린이병원 환자는 입원 기간 동안 1인당 **평균 6회** 이상 주사 기반 시술을 받고있으며, </br>
그 과정에서 많은 불안함과 두려움이 수반됩니다. </br> </br>
**TinglAR** 는 이러한 환아들을 위한 **주사바늘 불안감을 해소하는 AR 기반 서비스** 입니다.
